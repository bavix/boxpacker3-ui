# boxpacker3-ui

Debug helper boxpacker3

### Usage

```sh
go install github.com/bavix/boxpacker3-ui@latest
```

Service start (default port `3000`):
```sh
boxpacker3-ui
```

If you want to run on a different port, then run like this:
```sh
PORT=4000 boxpacker3-ui
```
