# boxpacker3-ui

Debug helper boxpacker3

### Usage

```sh
go install github.com/bavix/boxpacker3-ui@latest
boxpacker3-ui
```

Default port `3000`. If you want to run on a different port, then run like this:
```
PORT=4000 boxpacker3-ui
```
